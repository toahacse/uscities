### Installation

```sh
git clone https://github.com/toahacse/uscities.git
```

```sh
php artisan key:generate
```

```sh
composer install
composer update
```

```sh
copy .env.example to .env and connect database 
```

```sh
enter cmd php artisan migrate 
```

```sh
php artisan optimize 
```

```sh
php artisan serve 
```
